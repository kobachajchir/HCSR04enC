Definición del tipo de dato TCRT_t para sensores infrarrojos reflectivos.

Este archivo define la estructura de un sensor TCRT (infrarrojo reflectivo), incluyendo sus estados, flags, y variables utilizadas para la calibración, lectura y detección de objetos mediante el ADC.
